# Kutumbpur High School Website

A modern, responsive website for Kutumbpur High School built with React and Tailwind CSS.

## 🚀 Features

- **Modern UI/UX Design**
  - Responsive layout for all devices
  - Smooth animations and transitions
  - Interactive components
  - Beautiful gradient effects

- **Key Sections**
  - Home page with hero section
  - About Us
  - Contact Information
  - Updates & News
  - FAQ Section
  - Student Portal
  - Parent Portal
  - Extracurricular Activities
  - Alumni Section
  - Library
  - Events Calendar

## 🛠️ Technologies Used

- **Frontend Framework**: React.js
- **Styling**: Tailwind CSS
- **Routing**: React Router
- **UI Components**: Flowbite
- **Icons**: Heroicons
- **Animations**: Custom CSS animations

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/OPTB-P1-kutumbpur-high-school.git
```

2. Navigate to the project directory:
```bash
cd OPTB-P1-kutumbpur-high-school
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```
